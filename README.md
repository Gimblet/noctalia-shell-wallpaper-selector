# Noctalia Shell Wallpaper Selector Vicinae Extension

This is a simple extension made by myself to easily change your noctalia-shell wallpaper.
This project is straightforward, light and does not have any hard dependencies. 

I believe that this extension could be easily adapted to other wallpaper providers such as hyprpaper.

This extension requires Noctalia Shell to be installed and be set to be the default wallpaper provider.

This extension was tested on Noctalia Shell 4.6.6 but lower versions should work. I am not in any way afiliated
with the Noctalia Shell team. This is just a simple extension that I pretty much made for myself.

In the future I plan to add more features like implementing the random wallpaper IPC call.
